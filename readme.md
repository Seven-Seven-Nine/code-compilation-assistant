### Code Compilation Assistant

Помощник компиляции кода для **C++**. Активируется при запуске в командной строке (*т.е. в терминале обыкновенном*), при запуске программы указывается директория с исходным кодом (*src*) и директория, в которую будет скомпилирован проект (*bin*). Также нельзя забывать и про названия скомпилированного файла.

В программу также вводится имя компилятора, через который будет происходить компиляция и версия языка C++.

Дополнительно, потом реализовать возможность создания текстового файла, в котором уже будут находиться все необходимые данные для компиляции проекта, по типу:
```
CompilationData:
  src=src
  bin=bin
  compile=clang++
  version=c++20
  name-file=MyProgram
```

### Примеры команды

Здесь будут парочка примеров команды, пока-что не определился, как назвать исполняемый файл помощника: сокращённо (**cca**) или полностью (**code-compilation-assistant**). Лично мне нравится полное имя, но в практичности первый вариант выигрывает.

Пример командной строки:
```
>> code-compilation-assistant --src=src --bin=dist --compile=clang++ --version=c++20 --name-file=MyProgram
>> cca --src=src --bin=bin --compile=clang++ --version=c++20 --name-file=MyProgram
```
